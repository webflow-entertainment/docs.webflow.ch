    ProbabilityUtil

    This package contains the ProbabilityUtil. The utility is a simple tool to get random game objects from one or more lists. A weighting can be added to the lists, which sets the probability. The higher the weighting compared to the other lists, the higher the probability of getting one of the objects from that list. The weighting must be set for each list, and the first list must have the highest weighting. Then the following list must have a lower weighting value.

    Prerequisites

    • Unity Version 2020.3.17f1 or higher

    • Unity Burst 1.6.0 or higher

    File Description

    Description of the files relevant for the function.

    Type Name Description
    C# Script ProbabilityUtil.cs Contains the main function of the tool and must be added as a component to a game object.
    C# Script GameObjectList.cs The GameObjectList class contains the list for the individual game objects.
    C# Script RandomUtil.cs A utility that returns a random number. The utility is optimized for performance.
    C# Script ProbabilityUtilEditor The editor script is used to customize the ProbabilityUtil in the Unity Inspector. A link to the API documentation is available.

    How To

    1. Create a new unity project named ProbabilityDemo.

    2. Import the Burst package from the package manger of unity (Link below). Choose the unity registry. After the import of the package restart Unity.

    3. Import the ProbabiltyUtil package from the unity asset store.

    4. Open the demo scene and run the demo scene.

    5. The demo scene generates five game objects and uses the ProbabilityUtil to select randomly one of them.

    6. Add your own prefabs to test and add more lists. Customize the demo scene as desired. Activate the debug log to get more information from the unity console.
